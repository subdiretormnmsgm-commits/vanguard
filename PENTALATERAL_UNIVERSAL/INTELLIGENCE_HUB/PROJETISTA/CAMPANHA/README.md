# INTELLIGENCE_HUB/PROJETISTA/CAMPANHA

> Pasta-elo do Motor de Inteligência de Nicho (ver `MOTOR_INTELIGENCIA_NICHO.md`).
> Criada em 2026-06-16 para fechar o ELO QUEBRADO (Achado 1 da análise cirúrgica dos atores).

## Propósito

Destino do **MATERIAL DE CAMPANHA** gerado pelo Projetista via NotebookLM (Ação 4 do system prompt
do Projetista): cards, posts, infográficos de campanha, roteiros de abordagem, sequências de mensagem.

O que circula no motor é o **PRODUTO VANGUARD — a entrega do Cowork**, robustecida pelo Executor.
O Projetista consome esse produto, comanda o NotebookLM a gerar o material e o entrega NO CHAT ao
Diretor (P-075). Após o veredito, o **Músculo grava aqui** (P-124).

Quem lê esta pasta: o **Embaixador Digital**, que consome **duas fontes** — a entrega do Cowork E
os produtos do Projetista (esta pasta) — para operar a campanha LinkedIn/Instagram e prospectar a
Vanguard.

## Regras

- Quem escreve: **somente o Músculo** (P-124), após veredito do Diretor. O Projetista nunca grava aqui.
- O que entra: cards/posts/infográficos/roteiros aprovados pelo Diretor.
- Quem lê: Embaixador Digital (seção "Material do Projetista" -> "Cards e posts",
  "Roteiros de abordagem", "Infográficos de campanha").
- Linguagem blindada (R-3): nenhuma menção a IA/ferramentas — "especialistas da Vanguard".
- Isolamento (P-059): zero dados de clientes ativos; casos anonimizados.

## Nome canônico

Este nome de pasta (`PROJETISTA/CAMPANHA`) é exatamente o que o system prompt do Embaixador Digital
já cita. NÃO renomear sem atualizar o prompt (prompt zero-touch).
