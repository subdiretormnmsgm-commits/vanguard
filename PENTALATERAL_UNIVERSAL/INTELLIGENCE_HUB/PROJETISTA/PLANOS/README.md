# INTELLIGENCE_HUB/PROJETISTA/PLANOS

> Pasta-elo do Motor de Inteligência de Nicho (ver `MOTOR_INTELIGENCIA_NICHO.md`).
> Criada em 2026-06-16 para fechar o ELO QUEBRADO (Achado 1 da análise cirúrgica dos atores).

## Propósito

Destino dos **PLANOS DE EXECUÇÃO** gerados pelo Projetista.

O que circula no motor não é sinal bruto: é o **PRODUTO VANGUARD — a entrega do Cowork**,
robustecida pelo Executor Cowork. O Projetista consome esse produto e gera o plano, entregando-o
NO CHAT ao Diretor (P-075). Após o veredito do Diretor, o **Músculo grava aqui** (P-124 — escrita
única no acervo; o Músculo é portão de governança, não autor).

Quem lê esta pasta: o **Embaixador Digital**, que consome **duas fontes** — a entrega do Cowork E
os produtos do Projetista (esta pasta) — para prospectar a Vanguard.

## Regras

- Quem escreve: **somente o Músculo** (P-124), após veredito do Diretor. O Projetista nunca grava aqui.
- O que entra: PLANO_EXECUCAO_[nicho]_v[N].md aprovado pelo Diretor.
- Quem lê: Embaixador Digital (seção 6 "Produto Definido" + seção 8 "Abordagem Blindada").
- Linguagem blindada (R-3): nenhuma menção a IA/ferramentas — "especialistas da Vanguard".
- Isolamento (P-059): zero dados de clientes ativos; casos anonimizados.

## Nome canônico

Este nome de pasta (`PROJETISTA/PLANOS`) é exatamente o que o system prompt do Embaixador Digital
já cita. NÃO renomear sem atualizar o prompt — a pasta foi criada para casar com o prompt existente
(prompt zero-touch).
