## CICLO PDCA DO PENTALATERAL

```
PLAN  → Discovery + DIRETRIZ do Gemini
DO    → Build do Músculo (1–5 dias)
CHECK → check_leis.ps1 + Sentinel + Auditor
ACT   → relatorio_evolutivo + próxima iteração
```

> Cada projeto completo = 1 ciclo PDCA.
> O sistema melhora porque o loop nunca fecha sem aprendizado documentado.
