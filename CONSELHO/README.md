# CONSELHO QUADRILATERAL — Pasta Central de Documentos

Esta pasta contém TODOS os documentos que o Diretor precisa para o ritual pós-versão.
Nunca procure em mais nenhum lugar — está tudo aqui.

---

## ESTRUTURA

```
CONSELHO/
├── NotebookLM/     → ficheiros para CARREGAR no NotebookLM
└── Gemini/         → comandos para COLAR no Gemini
```

---

## RITUAL PÓS-VERSÃO (ordem obrigatória)

### PASSO 1 — Carregar no NotebookLM
Abrir NotebookLM → Adicionar fontes → carregar os ficheiros de `NotebookLM/`:
- `MEMORIA_VXX.txt` (da versão que acabou)
- `relatorio_evolutivo_vXX.txt` (da versão que acabou)
- `DIRETRIZ GEMINI VXX.txt` (se recebeu do Gemini)
- Documentos permanentes (VANGUARD_INNOVATION_AUDIT, BUSINESS_RULES, PLANO_NEGOCIO)

### PASSO 2 — Colar no Gemini
Abrir `Gemini/COMANDO_GEMINI_VXX.md` → copiar tudo → colar no Gemini

### PASSO 3 — Receber os 4 blocos do Gemini
- Bloco 0: plano de negócio desta semana
- Bloco 1: análise estratégica
- Bloco 2: DIRETRIZ VXX+1 → guardar em `NotebookLM/DIRETRIZ GEMINI VXX+1.txt`
- Bloco 3: colar no NotebookLM → gerar Skill
- Bloco 4: colar no terminal Claude Code

### PASSO 4 — Guardar Skill gerada
Copiar Skill do NotebookLM → guardar em `.claude/skills/vanguard-vXX-[nome].md`
