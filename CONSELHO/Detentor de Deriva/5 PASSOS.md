5 PASSOS
Você viu no vídeo como o Obsidian + Claude Code cria um segundo cérebro alimentado por IA — pesquisa autônoma, notas interconectadas e tudo organizado automaticamente. Aqui tá o passo a passo completo pra você montar o seu.
🧠 Passo 1: Baixar o Obsidian
O Obsidian é o app onde seu conhecimento vai morar. Gratuito, offline, e os arquivos são seus — não ficam na nuvem de ninguém. É como um caderno infinito onde cada página pode linkar pra qualquer outra.
Dica: Disponível pra Mac, Windows e Linux. Instalação padrão — baixa, arrasta pra Aplicativos, pronto.

Baixar Obsidian (grátis)
📁 Passo 2: Criar e configurar seu Vault
Ao abrir o Obsidian, crie um "Vault" — é só uma pasta no seu computador. Dê um nome (ex: "Meu Segundo Cérebro") e escolha onde salvar. O vault é a pasta raiz do seu cérebro digital.
Dica: Não precisa instalar plugins do Obsidian agora. O Master Kit cuida de tudo.
💻 Passo 3: Abrir o Terminal na pasta do Vault
Vá até a pasta do vault no Finder, clique com botão direito e selecione "Abrir novo terminal nesta pasta". O Claude Code precisa estar DENTRO do vault pra funcionar. No Windows: abra o Explorer, clique na barra de endereço e digite cmd.
Dica: Se o terminal abriu na pasta errada, feche e abra de novo com botão direito na pasta correta.
⚡ Passo 4: Instalar o Obsidian Master Kit
No Claude Code, rode dois comandos: primeiro /plugin marketplace add melgarafael/obsidian-master-kit (adiciona a fonte), depois /plugin install obsidian-master-kit@obsidian-master-kit (instala). Em segundos os superpoderes aparecem.
Dica: Confira digitando / — se aparecer /obsidian-master-kit:init, tá instalado.

Ver guia completo passo a passo
🚀 Passo 5: Inicializar com /obsidian-master-kit:init
Rode o comando, responda 7 perguntas sobre quem você é e seus interesses, e o kit cria toda a estrutura: pastas, templates, memória personalizada, conhecimentos iniciais — tudo organizado e pronto pra usar. É como ter um decorador profissional organizando sua casa nova.
Dica: Depois de inicializar, toda vez que abrir o Claude Code na pasta do vault, ele já tem os superpoderes do Master Kit.
