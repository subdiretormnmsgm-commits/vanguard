# Skill: vanguard-v2
**Descrição:** Evolução da Vanguard Tech para a V2 (A Máquina de Vendas).

## 1. 🎯 Objetivo
Transformar os dados captados pelo Quiz na V1 em autoridade imediata, com o objetivo de converter o Lead logo após a conclusão do diagnóstico. O foco absoluto desta versão é construir a área logada administrativa (Cockpit), atualizar a interface para maior retenção e implementar o motor de inteligência de vendas autônomo (Shadow Closer Lite) em Python.

## 2. 🧠 Leis do Conselho
- Siga estritamente a paleta Cyber-Premium e a filosofia de isolamento de escopo Vanilla JS (IIFE) já implementada na estrutura V1.
- **Retenção de UI:** Atualize a interface do Quiz com animações de "IA Processando" no momento do envio. O painel deve informar: "A nossa IA está a processar o seu relatório de gargalos... verifique o seu WhatsApp em 2 minutos.".
- **Cockpit (O Cérebro Visual):** Integre o Supabase Auth para proteger a nova rota `/dashboard`. Este será o ambiente onde o Diretor visualizará o "Mapa de Calor de Dores" das leads captadas e identificadas.
- **Automação Python (Shadow Closer Lite):** Crie o script `shadow_closer_v2.py` que deverá ler os Leads capturados na V1, calcular o "Score de Urgência" baseado na análise textual do gargalo e gerar de forma automatizada o ficheiro `diagnostico_lead.md` para cada cliente. Leads com "Gargalos de Escala" recebem prioridade VIP.

## 3. ⚙️ Execução
O workflow obrigatório é: `Brainstorm` -> `Plan` -> `Front-end & Code (Python)` -> `Code Review` -> `Git Commit`.
**Registo de Memória Obrigatório:** Documente a lógica matemática e semântica do algoritmo de pontuação criando o ficheiro `memorias/MEMORIA_02_INTELIGENCIA_VENDAS.md`.

## 4. 🔄 Auto-Propulsão e Insights LMM
